# 视频文件目录

将您的演示视频文件放在此目录中，并命名为 `demo.mp4`。

## 视频要求

- 格式：MP4
- 编码：H.264
- 分辨率：建议 1280x720 或更高
- 时长：建议 2-5 分钟
- 大小：建议小于 50MB（GitHub Pages 有文件大小限制）

## 如何添加视频

1. 将您的视频文件重命名为 `demo.mp4`
2. 将文件复制到此目录
3. 更新 `../images/video-poster.jpg` 为视频的封面图像
4. 重新部署网站

## 注意事项

如果您的视频文件较大，建议：

1. 压缩视频以减小文件大小
2. 或者将视频上传到 YouTube 或 Vimeo，然后在 `index.html` 中使用嵌入代码替换视频标签
